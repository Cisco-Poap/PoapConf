Poap Config Tool
Version : 0.1
Created Date : 20 Feb 2017
Steps to Use:
  1. Select platform,NXos version
  2. If not using dropdown you can enter values in the text box
  3. Enter the correct serial number for the switch
  4. Place the path where all the images are placed
  5. Provide the switch ip,gateway etc
  6. Fill details of the optional items if required
  7. Press Submit
  8. The required files gets created in the folder named toDir
  9. Copy all these files to USB and plugin to switch and reload items
  10. If anything more required contact Admin
